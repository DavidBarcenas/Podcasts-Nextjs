# Podcasts

Application to listen to podcast for free, made with react and nextjs

## Preview
![App screenshot](./.readme-static/app.png)

## Features

* SSR (Nextjs)
* mp3 player
* audioboom API

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
